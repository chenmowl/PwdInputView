---自定义属性：

    pwdLength:  密码个数

    borderColor:    边框颜色

    cursorColor:    光标颜色

    isCursorEnable: 是否开启光标

    isCipherEnable: 是否开启密文

    pwdPadding: 密码之间间隔

    borderWidth:    边框粗细或下划线粗细

    mode:   控件样式有两个, cheek边框样式、underline下划线样式
